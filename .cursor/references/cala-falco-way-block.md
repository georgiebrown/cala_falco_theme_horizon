# Cala Falco Way (PDP block)

- **Block type:** `cala-falco-way` (`blocks/cala-falco-way.liquid`), registered on `blocks/_product-details.liquid`.
- **Settings:** Image picker, heading (plain text, default `THE CALA FALCÓ WAY`), rich text body, padding.
- **Layout:** Mobile: image above copy, centered. Desktop: horizontal row, image left in a vertical oval frame (`border-radius: 50%`, `aspect-ratio: 3/4`), title uppercase with letter-spacing, body in `.rte` under the title, copy left-aligned on large screens.
- **Output:** Renders only if at least one of image, heading, or non-empty body is set.
